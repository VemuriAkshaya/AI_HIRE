 # AIHire

AIHire is an AI-powered job matching platform that helps users upload resumes and find suitable job opportunities based on their skills and qualifications.

## Features

* Resume Upload
* Skill Extraction
* Job Matching
* Match Score Generation

## Technologies Used

### Frontend

* React
  

### Backend

* Spring Boot
* Java
* Maven

### Database

* MySQL

## How to Run

### Backend

mvn spring-boot:run

### Frontend

npm install

npm run dev

## Workflow

1. User uploads resume
2. Skills are extracted
3. Jobs are matched
4. Match scores are generated
5. Results are displayed
